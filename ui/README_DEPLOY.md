# راهنمای سریع Deploy

## Build پروژه

```bash
cd ui
npm install
npm run build
```

## آپلود به دایرکت ادمین

1. تمام محتویات پوشه `dist` را به `public_html/panel` آپلود کنید
2. مطمئن شوید که فایل `.htaccess` هم آپلود شده است

## URL‌ها

- **پنل**: https://panel.vandapay.com/
- **API**: https://mr.vandapay.com/wp-json/cwm/v1

## بررسی

بعد از آپلود، به https://panel.vandapay.com/ بروید و مطمئن شوید که صفحه لاگین نمایش داده می‌شود.

برای جزئیات بیشتر، فایل `BUILD_INSTRUCTIONS.md` را مطالعه کنید.
